#프로그램 설명: 학생 성적 관리 프로그램 이 프로그램은 학생들의 성적을 입력받고, 총점/평균/학점을 계산해서 출력합니다.
또한 등수 계산, 성적 삭제, 검색, 총점 정렬 기능까지 포함되어 있습니다.
클래스(Student, StudentManager)를 사용했습니다.
기능 요약:
1. 학생 성적 입력
2. 전체 성적 출력
3. 등수 계산
4. 성적 삭제
5. 학생 검색
6. 총점으로 정렬
7. 평균 80점 이상 학생 수 확인
8. 프로그램 종료
